# Player Profile JSON System
This system manages a simple player profile using a JSON file as a tiny database.
It demonstrates all CRUD operations: Create, Read, Update, and Delete.

## Features
- Creates a JSON database (profile.json) with a player name and score
- Reads the stored profile and displays the data
- Updates the player’s score
- Deletes the score field from the database
- Confirms each operation clearly

# How to use
The system performs the following steps automatically:
- Create the JSON file with initial data
- Read and display the profile
- Update the score
- Delete the score

Each step writes changes back to profile.json.


## Learning Purpose
Practice with:
- JSON file handling (dump, load)
- CRUD operations
- overwriting and modifying JSON data
- simple state management using files
